# Books App

## A books app using NodeJS, MySQL, NextJS and Shadcn.

# Instructions

### Clone the Repo

```
git clone https://github.com/harshshukla2002/BooksApp
cd BooksApp
```

### For backend

```
cd backend
npm install
npm run dev
```

and server started at localhost:4001

### For Frontend

```
cd frontend
npm install
npm run dev
```

## and server started at localhost:3000
